## Color Scheme

- **Background**: Dark blue gradient (#0a0a1a â†’ #1a1a3a â†’ #0a0a1a)
- **Primary**: Purple/Blue gradient (#6366f1 â†’ #a855f7)
- **Text**: Light gray (#e0e0e0)
- **Accents**: Light blue (#a0a0ff)
- **Cards**: Semi-transparent dark blue with blur effect
- **Borders**: Subtle blue glow (rgba(100, 100, 255, 0.3))

## Key Visual Elements

1. **Animated Stars** - 100 white dots with twinkle animation
2. **Glassmorphism** - Transparent cards with backdrop blur
3. **Gradient Text** - Title uses purple-blue gradient
4. **Glow Effects** - Buttons and cards have subtle glow on hover
5. **Responsive Grid** - Auto-adjusts columns based on screen size
6. **Code Blocks** - Dark background with syntax highlighting colors

## Mobile Responsive

- Single column layout on screens < 768px
- Touch-friendly button sizes
- Readable text without zooming
- Optimized star animation for performance
